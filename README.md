# Ditiro Moabelo - Portfolio Website

A modern, responsive developer portfolio website with a "Kindness Era" pastel theme.

## 📁 File Structure

```
Ditiro Portfolio/
├── index.html
├── README.md
└── assets/
    ├── images/
    │   ├── myphoto1.jpg  (Hero section - blob-shaped portrait)
    │   ├── myphoto2.jpg  (Featured work section - background)
    │   └── myphoto3.jpg  (About me section - rounded rectangle)
    ├── css/
    │   └── style.css
    └── js/
        └── script.js
```

## 🖼️ Image Requirements

You need to add **three images** to the `assets/images/` folder:

1. **myphoto1.jpg** - Hero Section
   - Used in the blob-shaped container on the left side of the hero section
   - Recommended size: 800x800px or larger (square format works best)
   - Will be automatically cropped to fit the blob shape

2. **myphoto2.jpg** - Featured Work Section
   - Used as a subtle background overlay in the featured project section
   - Recommended size: 1920x1080px or larger (landscape format)
   - Will be blurred and have reduced opacity for aesthetic effect

3. **myphoto3.jpg** - About Me Section
   - Displayed in a rounded rectangle card beside your bio
   - Recommended size: 600x800px or larger (portrait format works well)
   - Will be displayed with rounded corners and shadow

## 🚀 Getting Started

1. **Add Your Images**
   - Place your three photos in the `assets/images/` folder
   - Name them exactly: `myphoto1.jpg`, `myphoto2.jpg`, `myphoto3.jpg`

2. **Open the Website**
   - Simply open `index.html` in your web browser
   - Or use a local server for best results

3. **Customize Content**
   - Edit `index.html` to update any text, links, or project details
   - Modify `assets/css/style.css` to adjust colors or styling
   - Update `assets/js/script.js` for additional interactivity
