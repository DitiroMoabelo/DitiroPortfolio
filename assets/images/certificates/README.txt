CERTIFICATE FILES PLACEMENT
============================

Please add your three certificate files to this folder with these exact names:

1. java-fundamentals.pdf (or .jpg/.png)
   - For: Java SA Fundamentals certificate

2. devops-fundamentals.pdf (or .jpg/.png)
   - For: PeopleCERT DevOps Fundamentals certificate

3. web-programming.pdf (or .jpg/.png)
   - For: Programming with HTML, CSS and JavaScript certificate

SUPPORTED FORMATS:
- PDF files (.pdf) - Recommended
- Image files (.jpg, .jpeg, .png)

The certificates will open in a new tab when clicked on the portfolio website.
